# Template: Workflow Onboarding

## Propósito

Template reutilizable para workflow agéntico de onboarding en código nuevo.

## Estructura

### PLAN - Exploración

```
Explorar proyecto/código para onboarding: [descripción]

OBJETIVO:
- Entender: [qué necesitamos entender]
- Aprender: [qué necesitamos aprender]

GENERA PLAN:
1. Módulos/componentes clave a explorar
2. Flujos principales a entender
3. Puntos de entrada identificados
4. Documentación a generar
```

### ACT - Análisis Asistido

```
Como Agente Onboarding, analiza:
- [Componente 1]: [qué entender]
- [Componente 2]: [qué entender]

Genera:
- Guía de arquitectura
- Flujos de trabajo
- Puntos de entrada
- Convenciones y patrones
```

### OBSERVE - Validación

```
Validar comprensión:
- [ ] Arquitectura entendida
- [ ] Flujos documentados
- [ ] Puntos de entrada claros
```

### REFLECT - Refinamiento

```
Reflexionar:
- ¿Qué falta entender?
- ¿Qué documentación necesita mejoras?
- ¿Qué ejemplos prácticos ayudarían?
```
