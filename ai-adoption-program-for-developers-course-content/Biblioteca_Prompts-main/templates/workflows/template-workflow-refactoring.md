# Template: Workflow Refactoring

## Propósito

Template reutilizable para workflow agéntico de refactoring seguro.

## Estructura

### PLAN - Análisis

```
Analizar código para refactoring: [módulo/función]

CÓDIGO:
[Código actual]

OBJETIVOS:
- [Objetivo 1]
- [Objetivo 2]

GENERA PLAN:
1. Análisis de código actual
2. Problemas identificados
3. Estrategia paso a paso
4. Tests como red de seguridad
```

### ACT - Ejecución Incremental

```
Paso 1: [Mejora pequeña]
- Cambio: [descripción]
- Validar: Tests pasan

Paso 2: [Siguiente mejora]
...

Continuar paso a paso.
```

### OBSERVE - Validación Continua

```
Después de cada paso:
- Tests pasan
- Comportamiento mantenido
- Mejora lograda
```

### REFLECT - Iteración

```
Reflexionar:
- Mejoras logradas
- Problemas encontrados
- Próximos pasos
```
